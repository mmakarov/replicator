REPLICATOR ДЛЯ WINDOWS 10/11

Как пользоваться:

1. Распакуйте архив в любую папку, например на Рабочий стол.
2. Запустите run-windows.bat двойным кликом.
3. Нажмите "Выбрать видео" и выберите один или несколько видеофайлов.
4. Нажмите "Выбрать аудио" и выберите один аудиофайл.
5. Оверлей overlay.png уже выбран из комплекта. Если нужен другой, нажмите "Выбрать оверлей".
6. Введите 4 текста и нажмите "Старт".

Готовое видео появится рядом с запускателем:

youtube_ready.mp4

Если аудио длинное, смотрите на лог: там будет идти прогресс ffmpeg.

Ничего устанавливать не нужно. В архив уже вложены приложение, Python, Qt, ffmpeg и шрифты.
